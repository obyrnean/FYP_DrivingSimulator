# Final Year Project - Ane O'Byrne
### Serious Games & Interactive Storytelling: Driving Education Simulation

## Overview  
This repository contains a **driving education simulation** developed as part of a Final Year Project exploring **serious games and interactive storytelling**.  

The project is centred around a **driving education case study**, where the simulation is used to demonstrate the potential of game-based systems as tools for **learning, engagement, and behavioural change**.  

Rather than focusing purely on entertainment, this simulation is designed to highlight how interactive experiences can be used to improve **driver awareness, hazard perception, and decision-making skills** in a safe and controlled environment.  

This repository documents the **design and development process** of the simulation, including:  
- Game assets (hand-drawn visuals)  
- C# scripts for gameplay mechanics and logic  
- Unity scenes and project structure  
- Supporting materials used in building the simulation  

---

##  Project Aim  
The aim of this project is to **demonstrate the effectiveness of serious games in education**, using driving safety as a practical application.  

It explores how:  
- Interactive storytelling can enhance user engagement  
- Simulated environments can replicate real-world scenarios  
- Player choices and consequences can reinforce learning outcomes  

---

## Features  
- Driving simulation scenarios based on real-world situations  
- Interactive interface
- Hand-drawn visual style for a unique and stylised experience  
- Immediate feedback on player decisions  
- Immersive audio-visual elements  

---
