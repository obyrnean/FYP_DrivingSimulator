# Scripts Folder Overview

This folder contains all the **C# scripts** used for the development and interactions in the **Driving Education Simulation**. The scripts are organized into subfolders based on their purpose to make it easy to understand the logic behind the simulation.

---

## Inspection Minigame
- This folder contains all scripts used for the **vehicle inspection mini-game** on the initial interface.  
- Scripts handle:  
  - Player interactions with the inspection system  
  - Functionality of arrows and visual guides  
  - Logic of the mini-game

---

## Level 1 Scene Scripts
- This folder contains scripts used for interactions and gameplay within **Level 1 of the simulation**.  
- Scripts handle:  
  - Player interactions with the environment  
  - Road sign recognition and responses  
  - Dialogue boxes and clickable elements  
  - Integration of mini-games within the level  

---

## Managers
- This folder contains all **manager scripts** that control overall game functionality.  
- Scripts handle:  
  - Panel and UI interactions  
  - Button functionality  
  - Audio and music behavior  
  - Scene transitions and management  
  - General game state management  

---

All scripts are written in **C#** for Unity and are organized to make the simulation modular, maintainable, and easy to extend.
