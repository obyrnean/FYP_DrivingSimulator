# Assets Folder Overview

This folder contains all the core resources used in the **Driving Education Simulation**. It is organized to make it easy to understand and locate the different types of assets used in the project.

---

## Audio
The `Audio` folder contains all the music and sound effects used in the simulation.  
- All audio sources are taken from [Pixabay Sound Effects](https://pixabay.com/sound-effects/) and are used to enhance immersion.  
- The folder includes:  
  - Background music  
  - Engine and vehicle sounds  
  - Collision and road effects  
  - UI click sounds

---

## Img
The `Img` folder contains all the **hand-drawn visuals and images** used in the game.  
- This includes:  
  - Backgrounds  
  - UI elements (buttons, panels, icons)  
- All visuals are designed specifically for the simulation to maintain a consistent hand-drawn art style.

---

## Scripts
The `Scripts` folder contains all the **C# scripts** used to implement the gameplay mechanics and simulation logic.  
- Scripts cover areas such as:  
  - Player controls 
  - Game management and scene transitions  
  - UI interactions  

---

This structure ensures that all resources for the simulation are **well-organized and easy to navigate**, making it clear how the simulation was built and what assets were used.
