# Img Folder Overview

This folder contains all the **hand-drawn and visual assets** used in the Driving Education Simulation. The images are organized into subfolders based on their use in the simulation, making it easy to understand the visual components of the project.

---

## Anatomy Inspect
- This folder contains all images used in the **CRA inspection mini-game**.  
- It includes:  
  - Arrows drawn in different colors  
  - Visual aids to guide player interaction during the mini-game  (blue = default, green = correct, red = incorrect)

---

## Level 1 Img
- This folder contains all images used in **Level 1 of the simulation**.  
- Assets include:  
  - Scene backgrounds  
  - Road signs and traffic signals  
  - Interactive buttons and clickable elements  
  - Dialogue boxes and UI overlays  

---

## Main Menu Interface
- This folder contains all images used in the **Main Menu interface**.  
- Assets include:  
  - Backgrounds for each menu panel  
  - Buttons and interactive elements  
  - Icons, logos, and decorative visuals  

---

All images in this folder are **hand-drawn** to maintain a consistent visual style across the simulation and enhance the player’s immersive experience.
